# LERTI - Aplicación web para aprendizaje de inglés

## Descripción
LERTI es una aplicación web diseñada y desarrollada para el aprendizaje de inglés mediante lecciones, ejercicios interactivos, repaso de errores y estadísticas.

## Tecnologías utilizadas
- React
- Vite
- React Router
- LocalStorage
- Chart.js

## Instalación y ejecución
1. Clonar el repositorio
2. Instalar dependencias:

npm install

3. Ejecutar en local:

npm run dev

4. Abrir en navegador la URL que aparece en terminal
